Random Quote Machine

This project was part of the FreeCodeCamp Front End Libraries Certification, which requires using AJAX to retrieve a random quote from any API of choice, and render it to screen with the ability to tweet it.

In this project, I'm using Quotes On Design API by Chris Coyier.

Interested in code? Check the project's repository on GitHub.
